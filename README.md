# Sparrow Atlas Editor

## Info

This is a tool for editing Sparrow/Starling atlas XML data without this being to do manually.	
This is a project that I made by learning C# and Visual Studio! So this can be lame or unfinished.	
You can contribute at all! Only if you want to make this thing possible to exists.

## Compiling

You will need:
- Visual Studio 2026 18.3.1 (18.3.2 is incompatible)
- Package "Desktop Development with .NET" (Optional ones can be disabled)
- .NET 10
